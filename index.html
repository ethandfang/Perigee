Data & method

Sources. Orbital elements and physical parameters are a one-time snapshot from the JPL Small-Body Database Query API, pulled 2026-07-04 at full precision. The close-approach list comes from the JPL CNEOS Close-Approach Data API (all known passes within 0.02 au through the end of 2033). Nothing on this page phones home — the dataset is a static file committed with the site.

What's plotted. Every potentially hazardous asteroid in the catalog, plus the largest other near-Earth asteroids ranked by absolute magnitude H. A PHA is defined as an object with Earth MOID ≤ 0.05 au and H ≤ 22.0.

Motion. Each object's position is computed live in your browser by two-body Keplerian propagation from its osculating elements at the catalog epoch: Kepler's equation is solved by Newton–Raphson for the eccentric anomaly, converted to true anomaly, and rotated into heliocentric ecliptic coordinates via the standard i/Ω/ω rotation. Planetary positions use JPL's approximate mean elements (1800–2050).

Caveats. Two-body propagation ignores planetary perturbations, so positions drift from truth over years — fine for a sky-wide picture, not for navigation. Close-approach dates and distances in the list are JPL's own solutions, not this page's propagation. 1 lunar distance (LD) ≈ 0.002569 au ≈ 384,400 km.

Refresh. Re-running scripts/fetch_data.mjs re-pulls the catalog and overwrites the snapshot.
